The classes that handles the core of the game.
The core of the game is everything which is Bomberman specific:
  • Object/classes representing the elements of Bombermen (bomberman, bomb, item, wall, fire etc.) placed in the model package (layer).
  • Classes that calculates/modifies the objects of the game.
  • Classes containing and implementing the game rules.
  • Classes that manages the rounds and the game running.
  • Classes that handles and manages inputs for the game.
  • Etc.
